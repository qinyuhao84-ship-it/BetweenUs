BetweenUs 隐私政策

正式发布前，请把主体名称、联系方式、备案号和正式域名替换成真实信息，并完成法律审核。

1. 我们收集什么

  1. 手机号：用于验证码登录、绑定账号、找回账号。
  2. Apple 账号标识：用于 Sign in with Apple 登录。
  3. 录音内容：用于生成冲突复盘报告。
  4. 转写文本与复盘结果：用于历史查看、再次阅读和账号同步。
  5. 购买信息：仅包括 Apple 交易号、商品 ID、购买状态和对应额度，不保存银行卡或支付卡信息。
  6. 诊断信息：用于排查故障与改进稳定性。

2. 我们如何使用这些信息

  1. 完成登录与身份验证。
  2. 生成复盘报告并保存历史记录。
  3. 校验 Apple IAP 支付并发放次数额度。
  4. 处理退款、撤销、客服支持和账号删除。
  5. 保障系统安全，识别滥用和异常访问。

3. 音频如何处理

  1. 录音会先进入后端进行转写处理。
  2. 为了让火山 ASR 读取音频，系统会把文件临时上传到火山 TOS。
  3. 转写完成后，原始音频会被删除，不作为长期资产保留。
  4. 长期保留的是转写文本、结构化报告和必要账务数据。

4. 我们如何共享信息

  1. 与短信服务商共享手机号和验证码发送所需信息。
  2. 与火山引擎共享音频处理所需信息。
  3. 与 DeepSeek 或等价模型服务共享生成报告所需文本。
  4. 在法律法规要求、监管要求或用户主动授权的情况下进行披露。

5. 你的权利

  1. 查看你的账号资料。
  2. 绑定或更换登录方式。
  3. 删除账号。
  4. 要求了解我们保存了哪些与你有关的数据。
  5. 通过客服渠道提出隐私相关问题。

6. 删除账号后会发生什么

  1. 用户资料会被删除。
  2. 历史复盘、额度、验证码记录和未完成任务会被清理。
  3. 与 Apple 登录相关的 refresh token 会被撤销。
  4. 删除后数据不可恢复。
