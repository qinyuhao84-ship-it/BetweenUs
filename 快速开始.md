# BetweenUs 快速开始

这份文档是给产品、运营、测试同学用的。
目标不是讲原理，而是让你最快看到 App 跑起来。

## 1. 启动后端

先开一个终端：

```bash
cd "/Users/qyh/Desktop/daily file/VibeCoding/BetweenUs/backend"
uv sync --dev
cp -n .env.example .env
redis-server
```

再开第二个终端：

```bash
cd "/Users/qyh/Desktop/daily file/VibeCoding/BetweenUs/backend"
uv run celery -A app.workers.celery_app.celery_app worker --loglevel=info
```

再开第三个终端：

```bash
cd "/Users/qyh/Desktop/daily file/VibeCoding/BetweenUs/backend"
uv run uvicorn app.main:app --reload --port 8000
```

看到 `http://127.0.0.1:8000/healthz` 返回 `ok`，就说明后端起来了。

## 2. 启动 iOS App

```bash
cd "/Users/qyh/Desktop/daily file/VibeCoding/BetweenUs/ios"
xcodegen generate --spec project.yml
open BetweenUs.xcodeproj
```

在 Xcode 里直接点运行。

## 3. 你应该怎么验

### 登录

1. 登录页会同时看到 `Apple 登录` 和 `手机号登录`。
2. 审核场景优先走 Apple 登录。
3. 正常用户可以走手机号验证码登录。
4. Apple 登录后，也可以在设置页绑定手机号。

### 核心功能

1. 点击“开始录音”。
2. 说几句话后结束录音。
3. 等待进度从录音、转写、分析一路完成。
4. 打开报告页，确认内容和你刚才说的话相关。
5. 到“历史”页确认这条记录已经保存。

### 充值

1. 打开设置页，进入“充值中心”。
2. 页面会展示 Apple IAP 次数包。
3. 真机联调时，商品价格以 App Store Connect 为准。

### 删除账号

1. 设置页点“删除账号”。
2. 确认后，重新进入 App 不应再看到原来的历史和额度。

## 4. 发布前必须补齐的真实资源

1. Apple 登录配置。
2. App Store Connect 的 IAP 商品。
3. 阿里云短信签名、模板和审核。
4. 火山 TOS + 火山 ASR。
5. DeepSeek API Key。
6. 已备案的域名与 HTTPS。
7. App 备案号与中国大陆上架信息。
