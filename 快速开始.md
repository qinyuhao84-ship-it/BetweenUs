# BetweenUs 快速开始（非技术版）

## 1) 启动后端服务（先做）

在终端打开项目后，依次执行：

```bash
cd "/Users/qyh/Desktop/daily file/VibeCoding/BetweenUs/backend"
uv sync --dev
cp -n .env.example .env
redis-server
```

新开一个终端窗口，执行：

```bash
cd "/Users/qyh/Desktop/daily file/VibeCoding/BetweenUs/backend"
uv run celery -A app.workers.celery_app.celery_app worker --loglevel=info
```

再新开一个终端窗口，执行：

```bash
cd "/Users/qyh/Desktop/daily file/VibeCoding/BetweenUs/backend"
uv run uvicorn app.main:app --reload --port 8000
```

看到 `Uvicorn running on http://127.0.0.1:8000` 就表示后端成功。

## 2) 启动 iOS App

```bash
cd "/Users/qyh/Desktop/daily file/VibeCoding/BetweenUs/ios"
xcodegen generate --spec project.yml
open BetweenUs.xcodeproj
```

在 Xcode 里点运行（Run）。

## 3) 你应该怎么验证

1. 首次启动会先看到登录页，输入手机号，点击“获取验证码”，在开发环境可直接看到页面上的验证码提示。
2. 输入验证码后点击“登录”，进入主页面。
3. 打开“复盘”页，点击“开始录音”，说几句话，再点“结束录音并开始复盘”。
4. 进度会从 0% 往上走，最后到“分析完成”。
5. 点“查看最新复盘报告”，应该看到摘要、转写片段、潜在诉求、修复建议、行动任务。
6. 切到“历史”页，应该能看到刚才那条记录。

## 4) 如果要切到真实 AI（可选）

默认是 `AI_PROVIDER_MODE=auto`，不配密钥也能跑通。
如果你要真实 ASR + 真实 LLM，把 `backend/.env` 里这两个值填上：

```bash
ASR_API_KEY=你的ASR密钥
LLM_API_KEY=你的LLM密钥
```

填完后重启 `worker` 和 `uvicorn` 即可生效。
